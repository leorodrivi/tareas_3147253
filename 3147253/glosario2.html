Glosario de TI

N O P Q R S T U V W X Y Z

N

Nube Híbrida:

Entorno de computación que combina una infraestructura local (nube privada) con servicios de nube pública.

Nodo:

Punto de conexión dentro de una red.

O

Open Source (Código Abierto):

Software cuyo código fuente está disponible públicamente para ser utilizado, modificado y distribuido.

Optimización SEO (Optimización para Motores de Búsqueda):

Proceso de mejorar la visibilidad de un sitio web en los resultados de los motores de búsqueda.

P

Protocolo:

Conjunto de reglas que rigen la comunicación entre dispositivos en una red.

Python:

Lenguaje de programación de alto nivel, interpretado y de propósito general, popular en ciencia de datos y desarrollo web.

Q

Q#:

Un lenguaje de programación cuántico desarrollado por Microsoft

QoS (Quality of Service):

Se refiere a la calidad de servicio de una red, y conceptos relacionados con "querie" (consulta) o "queue" (cola) en bases de datos y algoritmos.

R

Router:

Dispositivo de red que reenvía paquetes de datos entre diferentes redes.

Realidad Virtual (RV):

Tecnología que crea un entorno inmersivo generado por computadora al que el usuario puede acceder a través de dispositivos especiales.

S

Software:

Conjunto de programas, datos e instrucciones que permiten a un sistema informático realizar tareas específicas.

Servidor:

Ordenador o programa diseñado para proporcionar servicios a otros ordenadores o programas (clientes) en una red.

T

TCP/IP (Protocolo de Control de Transmisión/Protocolo de Internet):

Conjunto de protocolos de comunicación fundamentales para internet.

Tecnología:

Conjunto de conocimientos y herramientas utilizados para resolver problemas y facilitar la vida humana.

U

URL (Localizador Uniforme de Recursos):

Dirección web única que identifica un recurso en internet.

UX (Experiencia de Usuario):

Diseño de productos y servicios digitales centrado en la facilidad de uso, la accesibilidad y la satisfacción del usuario.

V

VPN (Red Privada Virtual):

Extensión segura de una red local a través de una red pública, como internet.

Videojuego:

Software diseñado para el entretenimiento interactivo a través de una interfaz visual.

W

WAN (Red de Área Amplia):

Red de comunicación que cubre una gran área geográfica, como internet.

Wi-Fi:

Tecnología de red inalámbrica que permite a los dispositivos conectarse a internet o a otras redes.

X

XML (Lenguaje de Marcado Extensible):

Lenguaje de marcado diseñado para transportar y almacenar datos.

XaaS (Todo como Servicio):

Término general que se refiere a la creciente variedad de servicios de computación disponibles a través de la nube.

Y

YouTube:

Plataforma para compartir videos en línea.

Yield (Rendimiento):

En publicidad digital, métrica que mide los ingresos generados por un activo digital durante un período específico.

Z

Zip:

Formato de archivo utilizado para comprimir uno o varios archivos en un solo archivo para facilitar su almacenamiento y transferencia.

Zero-day exploit:

Vulnerabilidad de seguridad en un software que es desconocida para el proveedor, lo que la hace potencialmente peligrosa ya que no hay una solución disponible.
