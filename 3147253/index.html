A B C D E F G H I J K L M

A

Algoritmo:

Conjunto de reglas o instrucciones ordenadas que permiten resolver un problema o realizar una tarea.

API (Interfaz de Programación de Aplicaciones):

Conjunto de definiciones y protocolos que permite la comunicación entre diferentes componentes de software.

B

Base de Datos:

Conjunto organizado de datos relacionados entre sí, almacenados para facilitar su acceso y gestión.

Blockchain:

Tecnología de registro distribuido y seguro que almacena transacciones en bloques enlazados criptográficamente.

C

Ciberseguridad:

Práctica de proteger sistemas, redes y datos digitales contra accesos no autorizados, ataques y daños.

Cloud Computing (Computación en la Nube):

Modelo de prestación de servicios tecnológicos a través de internet, como almacenamiento, software y potencia de cómputo.

D

Data Science (Ciencia de Datos):

Campo interdisciplinario que utiliza métodos científicos, procesos, algoritmos y sistemas para extraer conocimiento e insights a partir de datos.

Deep Learning (Aprendizaje Profundo):

Subcampo del aprendizaje automático que utiliza redes neuronales artificiales con múltiples capas para analizar datos complejos.

E

Ethernet:

Estándar de red de área local (LAN) ampliamente utilizado para conectar dispositivos en una red cableada.

E-commerce (Comercio Electrónico):

Compra y venta de bienes y servicios a través de internet.

F

Firewall:

Sistema de seguridad de red que controla el tráfico entrante y saliente, bloqueando accesos no autorizados.

Front-end:

Parte de una aplicación web o software con la que interactúa directamente el usuario (interfaz de usuario).

G

Gigabyte (GB):

Unidad de medida de almacenamiento digital, equivalente a 1024 megabytes.

GPS (Sistema de Posicionamiento Global):

Sistema de navegación basado en satélites que proporciona información precisa sobre la ubicación.

H

Hardware:

Componentes físicos de un sistema informático, como el procesador, la memoria y los dispositivos de entrada/salida.

HTML (Lenguaje de Marcado de Hipertexto):

Lenguaje de marcado estándar para crear páginas web y aplicaciones web.

I

Inteligencia Artificial (IA):

Campo de la informática que busca desarrollar sistemas capaces de realizar tareas que normalmente requieren inteligencia humana.

Internet de las Cosas (IoT):

Red de objetos físicos ("cosas") que están integrados con sensores, software y otras tecnologías para recopilar e intercambiar datos.

J

Java:

Lenguaje de programación orientado a objetos ampliamente utilizado en el desarrollo de aplicaciones empresariales y móviles.

JavaScript:

Lenguaje de programación interpretado que se utiliza principalmente para añadir interactividad a las páginas web.

K

Kernel:

Núcleo del sistema operativo que gestiona los recursos del hardware y proporciona servicios a las aplicaciones.

Keyword (Palabra Clave):

Término específico que los usuarios introducen en los motores de búsqueda para encontrar información.

L

LAN (Red de Área Local):

Red de comunicación que conecta dispositivos dentro de un área geográfica limitada, como una oficina o un hogar.

Linux:

Familia de sistemas operativos de código abierto basados en el kernel de Linux.

M

Malware:

Software malicioso diseñado para dañar o infiltrarse en sistemas informáticos sin el consentimiento del propietario.

Machine Learning (Aprendizaje Automático):

Subcampo de la inteligencia artificial que permite a los sistemas aprender de los datos sin ser programados explícitamente.
